~ 制作物Ⅰ：ポートフォリオサイト（本サイト） ~

  • ★概要：自己紹介やスキル、制作物を紹介するためのWebサイトです。「シンプルかつ直感的な操作性」をテーマに制作しました。
  • ★使用技術：HTML / CSS / Canva / VScode / GitHub
  • ★制作期間：約2週間
  • ★こだわりポイント：
    • レスポンシブ対応：PCだけでなくスマホやiPadにも最適化。スマホ時はハンバーガーメニューに切り替わる設計です。
    • UI/UXの追求：常に操作できるようメニューを上部に固定。ホバーアクション（色の変化）を加え、直感的な操作を促します。
    • デザイン：ピンクとグレーを基調に、シンプルさと親しみやすさを両立します。
    • 視覚的な工夫：レーダーチャートを用い、スキルを視覚的に表現しました。
    • グローバル対応：日本語だけでなく、中国語での閲覧にも対応しています。

~ 制作物Ⅱ：漢字検定ゲーム ~

  • ★概要：Java (JSP/Servlet) を用いて開発した、ブラウザ上で動作する漢字検定クイズゲームです。データベースと連携し、本格的な学習ツールとしての実用性を重視しました。
  • ★使用技術：Java / JSP / Servlet / MySQL / HTML / CSS / Eclipse / GitHub
  • ★制作期間：約2週間
  • ★こだわりポイント：
    • バックエンド設計：MySQLに問題を保存。SQLによるデータ管理を行うことで、問題の追加や難易度変更が容易な拡張性の高い設計にしました。
    • MVCモデルの意識：適切なパッケージ・フォルダ分けを行い、メンテナンス性の高いコードを心がけています。
    • 本格的なコンテンツ：実際の漢検過去問から抜粋。3回ミスでのゲームオーバー機能や、全問正解時の特別リザルト画面など、ゲーム性を高める工夫をしました。
    • UI/UX：和風をテーマにしたデザイン選定と、どの画面からでも即座にスタート画面に戻れるナビゲーションを実装しました。

スキル

現在学習中のITスキルや、これまで業務で使用してきたスキルをご紹介します。スキルの評価基準は以下の通りです。

  • ⭐0 完全初心者レベル：コードに触れたことがない。/操作経験が全くない。
  • ⭐1 入門レベル：サンプルを少し動かした程度。/基本操作を軽く触った程度。
  • ⭐2 初歩レベル：基礎構文を理解し、見本を真似できる。/基本操作を理解し、簡単な作業を再現できる。
  • ⭐3 基礎レベル：簡単な機能を自力で実装できる。/基本機能を問題なく使いこなせる。
  • ⭐4 実務レベル：実務で開発し、改善にも対応できる。/実務で安定して使え、改善もできる。
  • ⭐5 プロレベル：設計〜最適化まで高度に対応できる。/複雑な運用でも効率的に扱える。
