~ 作品Ⅱ：日语汉字检定游戏 ~

  • ★项目概述：使用Java (JSP/Servlet) 开发的网页版日语汉字检定问答游戏。通过与数据库联动，实现了作为专业学习工具的实用性。
  • ★使用技术：Java / JSP / Servlet / MySQL / HTML / CSS / Eclipse / GitHub
  • ★制作时间：约2周
  • ★设计亮点：
    • 后端设计：使用MySQL存储题目。通过SQL进行数据管理，使添加题目和修改难度变得非常简单，具有很强的扩展性。
    • MVC模式：遵循MVC设计思想进行合理的包（Package）和文件夹划分，确保代码的高可维护性。
    • 专业内容：题目摘录自实际的汉字检定历年真题。加入了3次错误强制结束、合格时的特殊结算页面等游戏化元素。
    • UI/UX：采用和风主题设计，并实现了从任何页面都能快速返回初始页面的导航功能。
