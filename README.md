# Portfolio Website
サイト：https://apple2026-01.github.io/Mizuta-Portfolio/

## 概要
自身のスキルを視覚化し、これまでの学習成果を一つの場所にまとめたWebサイトです。
PC、タブレット、スマートフォンのいずれから見ても最適なレイアウトで表示されるよう、フルレスポンシブで構築しました。

## 実装・機能のポイント
- **レスポンシブデザイン**: メディアクエリを活用し、デバイスサイズに応じた最適なレイアウト（ハンバーガーメニュー等）を実装。
- **UI/UXの工夫**: 
    - メニューバーの固定によるアクセシビリティの向上。
    - ホバーエフェクトを用いた、直感的なクリック誘導。
    - ラジオボタンとCSSのみを用いた、JavaScriptを使わない軽量な画像スライダーの実装。
- **スキルの可視化**: グラフ（レーダーチャート）を用い、現在のスキルセットを直感的に把握できるよう工夫。
- **多言語対応**: グローバルな視点を持ち、日本語だけでなく中国語での自己紹介・コンテンツも掲載。

## 使用技術
HTML / CSS / GitHub / Canva / VScode

## デザインのこだわり
- **配色**: グレー（誠実さ・シンプル）とピンク（親しみやすさ・自分らしさ）をメインカラーに採用。
- **一貫性**: サイト全体を通して余白を活かし、情報が煩雑にならないよう整理されたレイアウトを意識しました。
